Pages

This is a very simple example of a website built on Cloudflare Pages:

  • Cloudflare Pages you can deploy dynamic front-end applications in record time.
  • Pages is super fast, always up to date.
  • Deploy your project by connecting to your Git provider, uploading prebuilt assets directly to Pages with Direct Uploads or using Wrangler from the command line.
  • Build fast sites. In record time.
  • Cloudflare Pages is a JAMstack platform for frontend developers to collaborate and deploy websites.

Cloudflare's Network

Cloudflare Pages, like everything on Cloudflare, is supported by our massive network. That allows your website to load extremly fast by being delivered closer to your end users.
